Note that this is a little bit different from creating account/login. This time we need to APPEND PARAMETER STRING TO THE END OF URL in order to use GET method. And the urlParameters should be only the content of the file.
And here is the function just in case you do not have it (I did not wrote this. This is from StackOverflow):
