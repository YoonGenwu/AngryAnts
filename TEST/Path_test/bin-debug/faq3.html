Game Troubleshooting FAQ



First, ensure your operating system, Java SE, and browser configuration is supported with these compatibility charts.
Second, as a rule of thumb, keep Java up to date and remove older versions when updating.
I installed Java but it's not being recognized.
Run this test and refer back to the compatibility charts.
Mac OS X: Make sure you have the correct version of Java set as the default.
Ubuntu: http://www.webupd8.org/2012/09/install-oracle-java-8-in-ubuntu-via-ppa.html
I'm not sure if my Operating System is 32-bit or 64-bit.
Windows: use this guide.
Mac OS X: All supported versions are 64-bit.
Linux: Run the command 'uname -m' from the terminal.
The game won't start in Safari or Firefox on my Mac.
Make sure that your browser is not being run in 32-bit mode.
The game starts but once I click a video, I can still see the game but not the video.
If you're running Windows XP or Vista, refer to the JavaFX Media section in this article.

Click here to run the experimental Flash version of the game.
